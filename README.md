# GenAI Sales Analytics Chatbot (Microsoft Fabric)

## 🚀 Overview
This project builds a GenAI-powered chatbot that allows users to query sales data using natural language.

## 🧠 Features
- Natural language to SQL conversion
- Spark SQL execution on Fabric Lakehouse
- Optimized unified sales table
- Safe query generation

## 🏗️ Architecture
User → OpenAI → SQL → Spark → Lakehouse → Result

## 🛠️ Tech Stack
- Microsoft Fabric
- PySpark
- Spark SQL
- OpenAI API
- Delta Lake

## 📊 Example Queries
- Total sales by sales type
- Sales trend by year
- Top products by revenue

## 🔐 Security
- API keys handled via environment variables
- SQL validation implemented
